BizHawk Shaders
===============

These shaders are intended to run on BizHawk, for use with the DirectX Renderer


INSTALL & CONFIGURE
===================

1) Get the Shaders
2) Copy them into the \shaders directory located at the root of the BizHawk executable folder, or any subdirectory of the shaders.
3) Run EmuHawk
4) Go to Menu Config>Display in the Scaling & Filtering Tab, under the Scaling filter group, choose User and click Select
5) Select the CGP file of your choice
6) Enjoy!


Current Shaders
===============


* Retroizer		This one simulates an old CRT with color ghosting, deformation and low-definition. 
			Requires a high resolution framebuffer though (x3,x4)
